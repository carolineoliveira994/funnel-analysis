

#  Análise de Conversão em Fintech

🚨 **96% dos usuários abandonam antes de adicionar um produto ao carrinho**

Este projeto simula um cenário de análise de produto em uma plataforma digital, com foco em comportamento do usuário e otimização do funil de conversão.

---

## 🎯 Objetivo

Analisar o comportamento dos usuários e identificar gargalos no funil de conversão, com foco em melhorar a progressão desde o interesse inicial até ações de maior intenção.

---

## 📊 Dados

Eventos simulados de interação do usuário:

* `view`: visualização de produto
* `cart`: intenção de compra
* `purchase`: conversão final

---

## 🔍 Principais Descobertas

* Apenas **3,6%** dos usuários avançam de visualização para carrinho
* **96% de abandono** na primeira etapa do funil
* A maioria dos usuários interage com poucos produtos (**mediana = 2**)

---

## 🧠 Insight

O principal problema não está na aquisição de usuários, mas na **conversão do interesse em ação**.

Os usuários demonstram engajamento inicial, mas não avançam para etapas de maior intenção, indicando fricção na tomada de decisão.

---

## 💡 Recomendações

* Melhorar a apresentação e percepção de valor dos produtos
* Criar incentivos (descontos, frete grátis)
* Reduzir fricção na ação de adicionar ao carrinho
* Personalizar recomendações com base no comportamento

## Objective

Analyze user behavior and identify bottlenecks in the conversion funnel of a digital platform.

## Dataset

Simulated user interaction events:

* view
* cart
* purchase

## Key Findings

* Only **3.6%** of users move from view to cart
* **96% drop-off** at the first stage of the funnel
* Most users interact with only a few products (median = 2)

## Insight

The main issue is not user acquisition, but the conversion of interest into action.

Users show initial engagement but fail to progress to high-intent actions.

## Recommendations

* Improve product presentation and perceived value
* Add incentives (discounts, free shipping)
* Reduce friction in adding items to cart
* Personalize recommendations
